# 📝 To-Do List App

A simple and interactive To-Do List web application built using **HTML, CSS, and JavaScript**. This app helps users manage daily tasks efficiently by adding, completing, and deleting tasks.

---

## 🚀 Features

- ✅ Add new tasks
- ✔️ Mark tasks as completed
- ❌ Delete tasks
- 💾 Data stored in browser (Local Storage)
- 🎨 Simple and user-friendly UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
